# 📝 ToDo App

A simple and responsive **ToDo List Application** built with **React** and **Vite**. It helps users organize daily tasks by allowing them to add, remove, and manage todos with a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Delete individual tasks
- 🗑️ Clear all tasks
- 🚫 Prevent duplicate tasks
- ⚠️ Empty input validation
- ⌨️ Add task using the **Enter** key
- 📱 Fully responsive design
- 🎨 Modern UI with React Icons

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- React Icons
